When a user creates a new folder, they should be able to browse and add stocks directly from their existing "My Watchlist" without having to manually search for each stock. The flow should be:

When creating or editing a folder, show a list of stocks already in "My Watchlist" that the user can tap to add instantly

Already added stocks should be clearly marked so the user doesn't add duplicates


4. Deleting Custom Folders
Users should be able to delete any folder they have created, with the following rules:

A delete option should be accessible from the folder (e.g. long press, edit mode, or a settings icon)
"My Watchlist" must be protected and cannot be deleted — the delete option should never appear for it
Before deleting, show a confirmation prompt so folders are not removed by accident (e.g. "Are you sure you want to delete this folder? The stocks inside will not be deleted")
